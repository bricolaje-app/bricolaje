# Bricolaje

<p align="center">
  <img src="assets/icon.png" alt="Bricolaje Logo" width="128" height="128" />
</p>

*Bricolaje* is a desktop application that accelaterates and supports terminal operations with AI.
It combines with `bj` command-line tool to provide  command suggestions and history management.

## Installation

### *Bricolaje* Desktop

The latest release can be downloaded from
[Releases](https://github.com/bricolaje-app/bricolaje/releases/latest)
(MacOS only).

### `bj` CLI

```bash
curl --proto '=https' --tlsv1.2 -LsSf https://github.com/bricolaje-app/homebrew-tap/releases/download/bj-v0.1.0/bj-installer.sh | sh
```

## Features

- Natural language to command suggestions
- Execute or copy suggested commands
- Support for multiple AI providers (GitHub Copilot, Ollama, etc.)

### Pro Features

- All features of *Bricolaje* Desktop
- Command history management, search, and detailed view
- AI-generated command explanations
